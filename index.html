Hur du uppdaterar jobblistan

Uppdateringsknappen ovan simulerar en uppdatering på din webbläsare, men för att faktiskt uppdatera jobblistan behöver du:

  1. Gå till ditt GitHub repository
  2. Klicka på fliken "Actions"
  3. Välj "Update Job Listings" workflow
  4. Klicka på "Run workflow" knappen

Jobblistan uppdateras också automatiskt varje dag kl. 03:00 UTC.
